Detailed Description

MatchGesture Class. This class receives current frames and uses the HAAR cascade trained xmls to match gesture. The matched gesture is converted to a character and then sent to the main window module for speaking and display. It is inherited publicly from QWidget so that it can be linked with other Qt modules using Signals and Slots.
