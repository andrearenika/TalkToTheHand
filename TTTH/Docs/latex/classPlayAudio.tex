\subsection{Detailed Description}
Play Audio class. This class is the one which connects the hardware audio device to the application and helps the user to control the audio volume and the application to send the audio stream to the audio device. This class is inherited publicly from the QObject so that it can be easily connected to the other Qt Widgets with the use of signals and slots.
