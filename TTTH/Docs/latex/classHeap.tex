\subsection{Detailed Description}
\subsubsection*{template$<$typename type$>$ class Heap$<$ type $>$}

\hyperlink{classHeap}{Heap} class. This is a templatized heap class which takes any object as the heap input(on which $>$ and $<$ operators are overloaded) and organizes them in the heap to keep the maximum at the top of the heap. Though a lot of functionalities are given in this class but this application requires only max heap to be calculated which will be used in the matching module to calculate maximum number of occurences of the matched character.
